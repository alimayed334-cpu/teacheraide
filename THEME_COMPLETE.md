## 📸 لقطات الشاشة المتوقعة:

### **الشاشة الرئيسية:**
```
┌─────────────────────────────┐
│ [☰] مساعد المعلم    [🧪][🔔] │ ← AppBar أسود
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ [🟡م] مرحباً، المعلم    │ │ ← بطاقة داكنة
│ │      نتمنى لك يوماً...  │ │
│ └─────────────────────────┘ │
│                             │
│ نظرة سريعة                  │
│                             │
│ ┌──────────┐ ┌──────────┐  │
│ │ [📚] 1   │ │ [👥] 5   │  │ ← بطاقات صفراء
│ │ الفصول   │ │ الطلاب   │  │
│ └──────────┘ └──────────┘  │
│                             │
│ ┌──────────┐ ┌──────────┐  │
│ │ [✓] 0%   │ │ [📝] 0   │  │ ← أخضر + أصفر
│ │ الحضور   │ │ الاختبارات│  │
│ └──────────┘ └──────────┘  │
└─────────────────────────────┘
```
