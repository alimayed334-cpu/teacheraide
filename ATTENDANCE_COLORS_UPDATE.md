td # 🎨 تحديث ألوان صفحة الحضور

## ✅ التعديلات المطبقة

### 1. منطقة التصنيف والبحث (اختيار الفصل والتاريخ)
**اللون الجديد:** أسود داكن `#0D0D0D`
```dart
Container(
  decoration: const BoxDecoration(
    color: Color(0xFF0D0D0D), // أسود داكن مثل لون الصفحة
  ),
)
```

### 2. ملخص الحضور (الإحصائيات)
**اللون الجديد:** رمادي داكن `#2D2D2D`
```dart
Container(
  decoration: BoxDecoration(
    color: const Color(0xFF2D2D2D), // رمادي داكن
    borderRadius: BorderRadius.circular(12),
  ),
)
```

### 3. قائمة الطلاب
**اللون الجديد:** رمادي داكن `#2D2D2D`
```dart
Container(
  decoration: const BoxDecoration(
    color: Color(0xFF2D2D2D), // رمادي داكن
    border: Border(
      bottom: BorderSide(
        color: Color(0xFF404040), // خط فاصل رمادي
        width: 1,
      ),
    ),
  ),
)
```

### 4. إزالة المستطيل حول الأسماء
**قبل:** كان هناك CircleAvatar حول الحرف الأول
**بعد:** تم إزالته بالكامل - الاسم مباشرة بدون إطار

```dart
// تم إزالة:
CircleAvatar(
  radius: 20,
  backgroundColor: AppTheme.primaryColor.withOpacity(0.1),
  child: Text(student.name.substring(0, 1)),
)

// الآن فقط:
Text(
  student.name,
  style: const TextStyle(
    fontSize: 16,
    fontWeight: FontWeight.w500,
    color: Colors.white,
  ),
)
```

### 5. الخط الفاصل بين الطلاب
**اللون:** رمادي `#404040`
**السُمك:** 1 بكسل
```dart
border: Border(
  bottom: BorderSide(
    color: Color(0xFF404040),
    width: 1,
  ),
)
```

### 6. صفحة سجل الحضور
**منطقة فلترة التاريخ:** أسود داكن `#0D0D0D`

## ✅ التحسينات

### قبل التعديل ❌
- منطقة التصنيف: رمادي فاتح `#1E1E1E`
- قائمة الطلاب: رمادي متوسط `#2A2A2A`
- CircleAvatar حول الحرف الأول
- خط فاصل خفيف جداً

### بعد التعديل ✅
- منطقة التصنيف: أسود داكن `#0D0D0D` (مثل لون الصفحة)
- قائمة الطلاب: رمادي داكن `#2D2D2D`
- لا يوجد CircleAvatar - الاسم مباشرة
- خط فاصل واضح `#404040`
- تباين أفضل للألوان
- تصميم أنظف وأبسط
